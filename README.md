## Getting Started

These instructions will help you set up and run the project on your local machine.

### Prerequisites

- Node.js: [Download and install Node.js](https://nodejs.org/)

### Installation

1. Clone the repository:

   ```bash
   git clone 
   ```

2. install node modules:

   ```bash
   npm install
   ```

4. Run:

   ```bash
    npm run dev
   ```
